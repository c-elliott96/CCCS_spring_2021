(b)
$S_3$ is not abelian. Example:
\begin{enumerate}[nosep]
\item[] Let $f:(2)(1 \ 3)$ and $g:(1)(2 \ 3)$
\item[] $f(g(x)) = (1 \ 3 \ 2)$ but $g(f(x)) = (1 \ 2 \ 3)$
\end{enumerate}

(e)
$S_n$ is not abelian for $n > 3$. Example: $S_6$ is not abelian. In fact, $S_n$ is non-abelian for $n \geq 3$.
